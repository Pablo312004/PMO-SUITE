Guia de Configuração Google OAuth
  1. Acesse console.cloud.google.com → Criar projeto
  2. APIs & Services → Credentials → Create OAuth 2.0 Client
  3. Redirect URI: http://localhost:3000/api/auth/google/callback
  4. Ative Google Calendar API em Library
  5. Preencha no .env: GOOGLE_CLIENT_ID e GOOGLE_CLIENT_SECRET
  6. Reinicie o servidor
